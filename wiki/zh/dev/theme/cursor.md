# cursor

**使用他人素材制作主题包前请确保你已经获得了原作者的授权!**

`cursor` 是存放鼠标指针的文件夹, 鼠标指针的结构如下.

| 文件名           | 说明                    |
| ---------------- | ---------------------- |
| aero_arrow.cur   | 正常                    |
| aero_busy.ani    | 忙碌                    |
| aero_working.ani | 程序启动                |
| aero_ew.cur      | 横向调节                |
| aero_ns.cur      | 纵向调节                |
| aero_nwse.cur    | 调整大小 1(左上到右下)   |
| aero_nesw.cur    | 调整大小 2(左下到右上)   |
| aero_move.cur    | 移动                    |
| aero_link.cur    | 链接                    |
| aero_unavail.cur | 禁止                    |
| aero_up.cur      | 候选                    |
