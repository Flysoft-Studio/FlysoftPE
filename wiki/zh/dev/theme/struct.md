# 主题结构

主题用于为 FlysoftPE 更换视觉样式.

一个主题一般由以下文件和文件夹组成.

| 文件名        | 是否为可选  |
| ------------- | ---------- |
| cursor        | 是         |
| icon          | 是         |
| root          | 是         |
| config.reg    | 是         |
| start.png     | 是         |
| wallpaper.jpg | 是         |
| package.cmd   | 否         |
