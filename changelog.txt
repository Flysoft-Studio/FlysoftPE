FlysoftPE 更新日志 (Changelog)

说明:
    标 "*" 的项目为重要更新内容

v1.0 Beta 2021-07-30:
FlysoftPE 的第一个版本

v2.0 Beta 2021-08-22:
新增:
    * 新增 FlysoftPE Hub, 提供 FlysoftPE 的配置, 安装, 更新, 驱动管理等服务
    * 新增对 Ventoy 的支持
    * 引入 FlysoftPE Auto 技术, 可将 Portable Apps 上的应用自动抓取并做成插件包
    * 引入全局变量概念
    * 引入颜色模式设置, 可选用亮色或暗色
    * 引入分辨率设置, 可调整为支持的分辨率
    * 支持任务栏自定义, 可随意调整任务栏样式
    * 支持壁纸自定义, 可选择 JPEG 格式的图片文件
    增加 FlysoftPE Hub 的右键菜单
    新增对 .NET 程序的支持
    新增分辨率更改 API
    
修复:
    修复了输入法启动时定位出错的问题
    修复了输入法可能存在的皮肤消失问题
    修复了 PE 加载器中可能存在的执行顺序错乱问题
    修复了开始菜单可能存在的项目全部消失问题
    
改动:
    将文件的编辑选项打开的程序从记事本换为 Notepad3
    将 OpenShell 的样式进行了调整
    将 PENetwork 版本从 B11 降为 B4
    修改了任务栏右键菜单, 移除无用菜单项

移除:
    移除了 Default 主题
    移除了 Honeyview, 可自行添加回来
    移除了 OpenShell 的其他样式

v2.1 Beta 2021-08-23:
新增:
    新增 Hub 驱动管理对 PE 内驱动的移除功能

修复:
    修复 Hub 中的任务栏编辑不生效的问题
    修复 Hub 中的写入向导无法识别 MBR 格式的 Ventoy U 盘
    
改动:
    让 Hub 中的部分文本意思更准确
    
v2.2 Beta 2021-08-24:
新增:
    * 新增对高 DPI 字体大小的自动调节功能
    新增 DPI 适配 API

修复:
    修复 Hub 驱动管理无法识别相机设备的问题
    修复 Hub 启动时总是会出现启动盘版本过低的提示
    修复分辨率设置重启后失效的问题
    
改动:
    为任务栏的托盘启用折叠
    添加回快速访问, 以防止桌面有时莫名其妙出现快速访问图标

v3.0 Stable 2021-08-28:
首个稳定版 FlysoftPE
新增:
    * 新增简易对话框 API
    * 新增错误排查提示
    * 新增安全模式(不会加载插件和驱动)
    新增 url.dll 让应用可以唤起浏览器
    新增 sort.exe 让批处理可以排序
    新增 DirectX 组件
    新增无法检测到启动盘的提示
    
v3.1 Beta 2021-09-04:
新增:
    新增默认音量调整

改动:
    架构调整
    修改了右键菜单, 移除无用菜单项
    EFI 启动现在应该不会出现提示了
    第三方程序获取到的环境变量应该是正确的了

v3.2 Beta -> Stable 2021-09-12:
新增:
    新增颜色匹配组件
    新增屏幕键盘组件
    新增媒体组件
    更改颜色模式后任务栏颜色也会随之变化 (需要 FlysoftPE Hub 4.0 支持)

修复:
    修复启动失败时显示的界面有一部分是乱码的问题

改动:
    * 开始按钮现已更改为新样式, 更接近 Windows 11 的开始按钮
    默认任务栏透明度修改
    右下角时钟现在可以显示秒了
    RAMOS 方案的内置插件已更改

v4.0 Beta 2021-10-02:
新增:
    * 新增全局环境变量 (env.cmd), 详见文档

v4.1 Stable 2021-10-23:
修复:
    修复默认任务栏开始按钮图片错误问题
    修复默认任务栏颜色错误问题

v5.0 Stable 2021-10-24:
新增:
    * 新增在 PE 内可直接加载插件, 无需重启
    * 插件可以相互依赖 (depend.txt), 详见文档

修复:
    修复加载插件的路径中带有空格, 插件无法导入注册表的问题
    修复加载 ZIP 插件的路径中带有空格, 插件无法加载的问题

改动:
    对 FlysoftPE Loader (main.cmd) 进行了架构调整

v5.1 Stable 2021-11-20:
新增:
    * 新增 Windows 11 任务管理器
    * 新增新版任务栏时钟和日历 (T-Clock)
    新增 VCRuntime

修复:
    修复 RAMOS 方案保存用户数据选项不生效问题
    移除了 Toolkit 内无用的 Part.zip

改动:
    因开始菜单用户头像在高 DPI 时会错位, 所以隐藏用户头像

v6.0 Stable 2021-11-30:
新增:
    * 新增控制面板属性界面
    * 精简控制面板, 隐藏无效设置项

修复:
    修复主题包内的动态光标不生效问题

改动:
    移除 PENetwork 注册表残留
    暂时禁用 T-Clock
